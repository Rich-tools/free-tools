# Free Tools Hub

This is a ready-to-upload static tools website for GitHub Pages.

## Included
- Homepage
- 15 working tools
- Local demo tracking dashboard (`admin.html`)
- Shared CSS and JavaScript files

## Important about user tracking
Because GitHub Pages is static hosting, it cannot save real visitor analytics to a private database by itself.

This package includes:
- local browser tracking for testing
- a dashboard page to view that local data
- clean `trackAction()` hooks inside tools

For real users, connect an external analytics service such as Google Analytics or Microsoft Clarity.

## Upload steps
1. Open your GitHub repository.
2. Upload all files and folders from this project.
3. Commit changes.
4. GitHub Pages will publish the site automatically.
