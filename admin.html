How to track real users

Because GitHub Pages is static hosting, it cannot save visitor data in a private database by itself.

Best free setup:

  1. Create Google Analytics 4 or Microsoft Clarity.
  2. Copy the tracking script they give you.
  3. Paste it inside the <head> of every page, or a common template if you later use a generator.

This project already includes a clean place for simple action tracking using trackAction('Action name').
